# ZeloCoreCMS

ZeloCoreCMS is the most powerful, secure, and developer-friendly open-source CMS in the world. Built for Builders. Loved by Editors. Trusted by Enterprises.

## About ZeloCoreCMS

ZeloCoreCMS is an open-source CMS that clones and massively improves upon WordPress's core, adds headless API architecture, real-time visual editing, built-in AI content assistance, enterprise-grade security, and a one-click WordPress migration tool — all for free, forever, with no vendor lock-in.

### Core Architecture Principles

1. **Universal Hostability:** Runs on shared hosting, VPS, cloud, Docker, or bare metal. Requires only PHP 8.2+ and MySQL/MariaDB. Zero Node.js required in production.
2. **Security at the Core:** Security constraints enforced at the architecture level, not the plugin level. Three-tier plugin isolation model covers every hosting type.
3. **Developer Experience = Product Quality:** Built on modern PHP 8.2+ and Laravel, taking advantage of Fibers, Enums, and modern language features.
